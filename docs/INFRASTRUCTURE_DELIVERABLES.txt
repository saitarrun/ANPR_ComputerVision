================================================================================
COST ANALYSIS
================================================================================

  Quick Wins (Save 20-30%)
    - Disable Multi-AZ for dev/stage (-50% RDS)
    - Use smaller instances (-60% RDS/Redis)
    - Reduce backup retention (-75%)
    - Enable S3 VPC Endpoint (-$32/month)

  Medium Effort (Save 40-50%)
    - Use Fargate Spot (-70%)
    - Auto-scale baseline instances (-50% off-peak)
    - Cache with CloudFront (-40% data transfer)
    - Compress data (gzip/WebP)

  Architectural (Save 50%+)
    - Use Aurora PostgreSQL (-30%)
    - Use ECS on EC2 (-50%, ops overhead)
    - Move to S3 Glacier + Athena (-80%)
    - Use Lambda for async tasks (-70%)
