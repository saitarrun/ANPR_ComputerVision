# ANPR Local Development Environment

**Production-grade local development setup in <30 seconds.**

Complete Docker Compose environment that mirrors production infrastructure, enabling developers to build, test, and debug without manual configuration.

## ⚡ Quick Start

```bash
# Clone repo
git clone <repo> && cd ComputerVision\ Project

# One-command setup (installs deps, starts services, seeds DB)
make dev-setup

# Verify everything works
make test

# View API docs
open http://localhost:8000/docs
```

That's it. You now have:
- ✅ PostgreSQL 16 database
- ✅ Redis 7 cache + task broker
- ✅ MinIO S3-compatible storage
- ✅ FastAPI backend with hot-reload
- ✅ Celery background worker
- ✅ pgAdmin database UI
- ✅ Redis Commander inspection UI

## 🐛 Troubleshooting

### Services won't start
```bash
# Check status
docker-compose ps

# View logs
docker-compose logs

# Restart
docker-compose restart
```

### Port already in use
```bash
lsof -i :5432          # Find process using port 5432
kill -9 <PID>          # Kill it
docker-compose up -d   # Restart
```

### Hot-reload not working
```bash
# Verify file mount
docker-compose exec api ls -la /app/api

# Restart if needed
docker-compose restart api
```

### Full reset (keep volumes)
```bash
docker-compose down
docker-compose up --build
python scripts/init_dev_env.py
```

### Nuclear option (delete everything)
```bash
make docker-nuke       # Remove containers + volumes
make dev-setup         # Full setup from scratch
```

**For more issues:** See `DOCKER_TROUBLESHOOTING.md`
